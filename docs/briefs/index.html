06 — Tes tâches post-launch
Prompts prêts à utiliser dans Cowork

Chaque carte = un prompt que tu copies dans Cowork. Il fait le travail, toi tu valides.

Semaine 1 — Lancement

Kit de lancement

Posts X, Reddit, LinkedIn, email beta testers. Tout le contenu de launch en un batch.

"Notre app FreedomTracker vient de sortir sur l'App Store. App de suivi de dettes pour Canadiens. Crée : 5 tweets thread, 1 post Reddit r/PersonalFinanceCanada, 1 post LinkedIn. Ton authentique. Sauvegarde dans marketing/launch-kit/"
→ Dossier avec tous les textes prêts
Semaines 1-4 — ASO

Optimisation App Store

Analyse tes positions Astro et ajuste descriptions + keywords.

"Voici l'export CSV d'Astro (dans astro-exports/). Analyse nos positions, identifie les keywords qui performent vs stagnent. Nouvelle version des keywords (100 chars) + description ajustée. Sauvegarde dans marketing/aso-v2.md"
→ Keywords + descriptions optimisés
Chaque semaine

Content calendar

Calendrier de contenu par plateforme, avec textes drafts.

"Content calendar 2 prochaines semaines pour CreditCopilot. Mix tips crédit Canada, behind-the-scenes, questions engageantes. Format spreadsheet. Sauvegarde dans marketing/content-w3.xlsx"
→ Spreadsheet dates + plateformes + textes
Bi-mensuel

Monitoring reviews

Résumé des reviews, tendances, réponses suggérées, bugs à remonter.

"Cherche les reviews récentes de CreditCopilot sur l'App Store CA et US. Résume positif/négatif. Propose des réponses. Liste bugs/features pour Martin. Rapport dans marketing/reviews-feb.md"
→ Rapport avec actions concrètes
Mensuel

Analyse compétitive

Check sur les compétiteurs : features, pricing, reviews.

"Analyse les 5 principaux compétiteurs de FreedomTracker sur l'App Store CA. Compare pricing, rating, features, plaintes. Tableau dans marketing/competitive-feb.xlsx"
→ Excel comparatif stratégique
À chaque update

Release notes

Transformer le changelog technique en notes user-friendly.

"Lis CHANGELOG.md. Génère release notes v1.3 — pas de jargon, ton motivant. Ce que les users vont aimer. Sauvegarde dans marketing/release-v1.3.md"
→ Notes prêtes pour App Store Connect

08 — C'est quoi Cowork
En 30 secondes

Claude qui travaille sur ton ordi. Pas un chat — un assistant qui agit sur tes fichiers.

Ce que Cowork fait

Tu lui donnes accès à un dossier sur ton Mac. Tu décris ce que tu veux. Il fait un plan, te le montre, et exécute. Documents, spreadsheets, recherche web, contenu — directement dans tes fichiers.

Ce que Cowork n'est pas

C'est pas un chatbot copier-coller. C'est pas un outil de code (ça c'est Martin). C'est pour le contenu, l'analyse, l'organisation — ton rôle dans l'équipe.
